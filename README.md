# Retail Sales Data Warehouse

## Overview
This project implements a retail sales data warehouse using SQL, PySpark, PostgreSQL, and AWS S3 for analytics reporting and ETL processing.

## Tech Stack
- SQL
- Python
- PySpark
- PostgreSQL
- AWS S3

## Architecture
AWS S3 → PySpark ETL → PostgreSQL → Reporting

## Features
- ETL pipeline development
- Star schema modeling
- Fact and dimension tables
- KPI reporting queries
- Data validation and cleansing
